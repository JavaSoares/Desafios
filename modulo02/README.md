## 📁 Como baixar o projeto

```bash

# Clonar o repositorio
$ git clone https://wwww.github.com/JavaSoares/Desafios

# Entrar no diretorio 
$ cd Desafios

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
